Subscriptions_body

  • event : String
    Event trigger
    Enum:
    document.created
    document.sent
    document.viewed
    document.reviewed
    document.signed
    document.approved
    document.expired
    document.rejected
    document.downloaded
    document.printed
    document.forwarded
    document.partially_signed
    document.commented
    document.hardbounced
    document.imported
  • target_url : String
    The URL to post payload to on event
  • host : String
    The host name to send the post to
  • payload : String
    Additional payload data to post
  • global : Boolean
    If the webhook should catch events for all entity users.
  • notification_email : String
    If the event fails, send a notification to this email instead of the sender
