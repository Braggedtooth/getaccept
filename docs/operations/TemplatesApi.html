This is the API

listFields

Receive a list of fields for a specific template.

URL

https://api.getaccept.com/v1/templates/{templateId}/fields

HTTP Method

GET

Response Type

Fields

Parameters

  • Path: templateId String

    Template ID

listRoles

Receive a list of roles for a specific template.

URL

https://api.getaccept.com/v1/templates/{templateId}/roles

HTTP Method

GET

Response Type

inline_response_200_5

Parameters

  • Path: templateId String

    Template ID

listTemplates

Receive a list of available document templates. You can add new templates under the Templates-section in GetAccept.

URL

https://api.getaccept.com/v1/templates

HTTP Method

GET

Response Type

inline_response_200_2

Parameters

  • Query: showall String

    Show all templates for entity

  • Query: folder_id String

    Show templates in a specific folder
