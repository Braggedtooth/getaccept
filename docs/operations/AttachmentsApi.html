This is the API

listAttachments

Receive a list of available attachments. You can add new attachments in GetAccept using a new document.

URL

https://api.getaccept.com/v1/attachments

HTTP Method

GET

Response Type

inline_response_200_11

Parameters

uploadAttachment

You can upload attachment files to include with documents you send. Successful request will return the attachment_id. Use attachment_id to assign an attachment to a document.

URL

https://api.getaccept.com/v1/upload/attachment

HTTP Method

POST

Response Type

inline_response_200_9

Parameters

  • Body: body Upload_attachment_body(Upload_attachment_body)

    A JSON object containing the uploaded attachment information
